##  封装

将数据和方法捆绑在对象内部，仅通过暴露的接口与外界交互，保护数据安全并简化使用。

## 案例解析

```python
class Foo:
    def __init__(self,name,age):
        self.name = name
        self.age = age

    def detail(self):
        print(self.name)
        print(self.age)

obj1 = Foo('chensong',18)
obj2 = Foo('aaron',16)

# 通过对象直接调用被封装的内容
print(obj1.name)
print(obj2.age)

# 通过 self 间接调用被封装的内容
obj1.detail()
obj2.detail()
```

### 摆放家具

```python
class House:
    def __init__(self,type,area):
        self.type = type
        self.area = area
        self.free_area = area
        self.houseitems = []

    def __str__(self):
        return f'种类:{self.type}  面积:{self.area}  剩余面积:{self.free_area}  房屋物品:{self.houseitems}'
    def add_houseitem(self,item):
        self.houseitems.append(item.name)
        self.free_area-= item.area


class HouseItem:
    def __init__(self,name,area):
        self.name = name
        self.area = area
    def __str__(self):
        return self.name

house = House('Box',50)
print(house)
item1 = HouseItem('TV',2)
item2 = HouseItem('bed',4)
house.add_houseitem(item1)
house.add_houseitem(item2)
print(house)
```
